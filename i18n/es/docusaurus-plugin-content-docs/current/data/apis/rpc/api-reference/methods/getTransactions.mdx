### Usar el Laboratorio

El método `getTransactions` devuelve una **lista paginada de transacciones** comenzando desde un punto definido por el usuario. Puedes continuar paginando a través de los resultados siempre que el rango solicitado esté dentro de la **ventana de retención de historial** del proveedor RPC correspondiente.

Usa este método en el [Laboratorio Stellar](https://laboratory.stellar.org) para explorar la actividad reciente dentro de un marco temporal o rango de libro mayor dado.

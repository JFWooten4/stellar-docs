# Operations & Transactions

<YouTube ID="sP1er2wIL-0" />

Stellar transactions bundle operations into atomic submissions, so you can move assets, authorize accounts, or run smart contracts without juggling multiple network calls. This section shows how a transaction travels from signing to the ledger and spotlights the operations you can mix and match to build your use cases.
